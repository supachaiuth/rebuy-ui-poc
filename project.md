Project: Apple Device Buyback Website UI POC

Current Phase:
UI-only POC

Goal:
Create a premium modern cinematic website UI prototype for a device buyback business.

Scope for this phase:
- 3 pages only
- UI only
- no backend
- no real logic
- no real submission flow
- buttons can be placeholders

Pages:
1. Home page
2. Device details input page
3. Summary / review page

Design Direction:
- premium
- modern
- cinematic
- luxury
- minimal
- dark warm neutral palette
- elegant whitespace
- refined typography
- subtle motion

Avoid:
- generic SaaS look
- crowded layouts
- too many cards
- bright gradients
- flashy animation
- too much text
